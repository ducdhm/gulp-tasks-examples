# Gulp Tasks Examples

## I. Require
 * Install NPM at [here](https://nodejs.org/en/download/)
 * Install `gulp-cli` at [here](https://www.npmjs.com/package/gulp-cli)
 
## II. Install packages
```
npm install
```
 
or
```
yarn install
```
 
## III. Run
### 1. Build CSS
`gulp build-css`

### 2. Build JS
`gulp build-js`
