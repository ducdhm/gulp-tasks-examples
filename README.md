# Gulp Tasks Examples

## Require
 * Install NPM at [here](https://nodejs.org/en/download/)
 * Install `gulp-cli` at [here](https://www.npmjs.com/package/gulp-cli)
 
 ## Install packages
 ```
 npm install
 ```
 
or
```
yarn install
```
 
 ## Run
 ### Build CSS
 `gulp build-css`
 
 ### Build JS
 `gulp build-js`
